# HerbMind V2 UI 视觉效果设计文档

## 1. 设计理念

### 1.1 整体风格
- **专业感**：简洁、清晰、信息密度适中
- **国风元素**：现代设计 + 中医传统配色
- **高效查询**：减少视觉干扰，突出搜索功能

### 1.2 设计原则
1. **层次清晰**：通过卡片分组区分信息层级
2. **留白适度**：内容区域适当留白，避免拥挤
3. **色彩克制**：以中性色为主，功能性颜色为辅

## 2. 色彩系统

### 2.1 主色调
| 颜色 | 色值 | 用途 |
|-----|------|------|
| 主色（竹青） | `#7CB342` | 主要按钮、选中状态、强调 |
| 主色深 | `#558B2F` | 主色 hover/pressed 状态 |
| 主色浅 | `#AED581` | 浅色背景、标签 |

### 2.2 中性色
| 颜色 | 色值 | 用途 |
|-----|------|------|
| 标题/正文 | `#2C2C2C` | 主要文字 |
| 次要文字 | `#666666` | 辅助说明、拼音 |
| 占位文字 | `#999999` | hint 文字 |
| 分隔线 | `#E0E0E0` | 分割线、边框 |
| 背景灰 | `#F5F5F5` | 页面背景、卡片背景 |

### 2.3 功能色
| 颜色 | 色值 | 用途 |
|-----|------|------|
| 错误 | `#E53935` | 错误提示 |
| 警告 | `#FFB300` | 警告提示 |
| 成功 | `#43A047` | 成功提示 |
| 链接 | `#1976D2` | 可点击链接 |

### 2.4 国风辅助色
| 颜色 | 色值 | 用途 |
|-----|------|------|
| 赭石 | `#8D6E63` | 次要强调、分类标签 |
| 宣纸白 | `#FAFAF8` | 页面背景 |
| 墨黑 | `#2C2C2C` | 主要文字 |
| 朱砂 | `#C62828` | 毒性药材标识 |

## 3. 字体规范

### 3.1 字体族
- **中文**：系统默认（苹方/思源黑体）
- **英文/拉丁名**：`Roboto`, `serif`（拉丁学名用斜体）
- **拼音**：`Roboto`（非斜体）

### 3.2 字号规范
| 层级 | 字号 | 字重 | 用途 |
|-----|------|------|------|
| H1 | 24sp | Bold | 页面标题 |
| H2 | 20sp | Bold | 卡片标题 |
| H3 | 18sp | Medium | 药材名称 |
| Body | 16sp | Regular | 正文内容 |
| Caption | 14sp | Regular | 辅助信息 |
| Small | 12sp | Regular | 标签、拼音 |

## 6. 交互规范

### 6.1 触摸反馈
- 按钮按下：透明度 0.7 或背景色变深
- 卡片按下：轻微缩小 scale(0.98)
- 响应时间：
  - 点击反馈：≤ 100ms
  - 页面跳转：≤ 300ms

### 6.2 动画规范
| 动画 | 时长 | 缓动函数 |
|-----|------|---------|
| 页面进入 | 300ms | ease-out |
| 页面退出 | 200ms | ease-in |
| 卡片展开 | 250ms | ease-in-out |
| 筛选切换 | 150ms | ease-out |
| 图片加载 | 淡入 200ms | ease-out |

### 6.3 手势支持
- 列表页：下拉刷新
- 详情页：左右滑动切换图片
- 卡片：长按显示更多选项

## 7. 适配规范

### 7.1 屏幕适配
- 基准宽度：360dp
- 网格列数：
  - 小屏（<360dp）：2列
  - 中屏（360-480dp）：2列
  - 大屏（>480dp）：3-4列

### 7.2 横屏适配
- 首页：搜索框上移，筛选区横向展开
- 详情页：左右分栏（左侧图片，右侧信息）

## 8. 图标系统

使用 Material Icons Outlined 风格：
- 搜索: `search`
- 返回: `arrow_back`
- 收藏: `star` / `star_border`
- 分享: `share`
- 筛选: `filter_list`
- 展开: `expand_more`
- 收起: `expand_less`
- 清除: `clear`
- 历史: `history`
