# HerbMind V2 产品需求文档 (PRD)

## 1. 项目概述

### 1.1 产品定位
HerbMind V2 是一款基于 HKBU（香港浸会大学）中药材数据库的专业中药材查询工具，帮助中医学生、从业者快速精准地查找药材和方剂信息。

### 1.2 核心目标
- **查询效率**：3秒内定位目标药材
- **信息完整**：提供 HKBU 标准的药材详情 + 方剂关联
- **体验流畅**：简洁专业的交互设计

### 1.3 数据来源
- **药材数据**：HKBU 中药材数据库，共 420 种药材
- **方剂数据**：HKBU 中药方剂数据库（与药材关联）
- **图片资源**：药材饮片图（concocted/）、植物图（plants/）

## 2. 功能需求

### 2.1 首页

#### 布局结构
```
┌─────────────────────────────┐
│  搜索框（支持拼音/中文/拉丁名）  │
├─────────────────────────────┤
│  快捷筛选区（横向滚动 Chips）  │
│  [类别] [产地] [性味] [归经]   │
│  [功效类别]                   │
├─────────────────────────────┤
│  最近查看（横向滚动卡片）       │
├─────────────────────────────┤
│  常用药材（按使用频率排序）     │
└─────────────────────────────┘
```

#### 功能点
1. **搜索框**
   - 支持中文、拼音、拉丁名搜索
   - 实时显示搜索建议（最多5条）
   - 支持扫码搜索（未来扩展）

2. **快捷筛选**
   - 类别：根及根茎类、果实及种子类、花类、叶类等12类
   - 产地：四川、江苏、浙江、河南等20+省份
   - 性味：苦、甘、辛、温、寒等
   - 归经：肺、脾、肾、肝、心等
   - 功效类别：解表、清热、补虚、理气等

3. **最近查看**
   - 显示最近浏览的10个药材
   - 支持清空历史

4. **常用药材**
   - 显示高频查询的药材（基于统计数据）
   - 按类别分组展示

### 2.2 搜索结果页

#### 布局
- 顶部：搜索框（保留当前搜索词）+ 已选筛选条件（可删除）
- 主体：药材列表（网格/列表双视图）
- 底部：分页加载

#### 药材卡片信息
- 药材图片（饮片图）
- 中文名 + 拼音
- 类别标签
- 主要性味（简写）

#### 排序选项
- 相关度（默认）
- 名称 A-Z
- 拉丁名 A-Z

## 3. 搜索逻辑

### 3.1 搜索维度
| 搜索类型 | 匹配字段 | 权重 |
|---------|---------|------|
| 名称搜索 | name, pinyin, latin_name, aliases | 100 |
| 功效搜索 | effects | 40 |
| 主治搜索 | indications | 30 |
| 产地搜索 | origin | 20 |
| 性味搜索 | nature, flavor | 20 |

### 3.2 智能识别
输入内容自动识别搜索维度：
- "四川" → 产地筛选
- "清热" → 功效筛选
- "苦" → 性味筛选

## 4. 数据模型

### 4.1 药材 (Herb)
```kotlin
data class Herb(
    val id: String,              // 唯一标识（如：awei）
    val name: String,            // 中文名
    val pinyin: String,          // 拼音
    val latinName: String,       // 拉丁名
    val aliases: List<String>,   // 别名
    val category: String,        // 类别
    val nature: String,          // 性味（如：苦、辛，温）
    val flavor: List<String>,    // 味（如：["辛", "苦"]）
    val meridians: List<String>, // 归经
    val effects: List<String>,   // 功效
    val indications: List<String>, // 主治
    val origin: String,          // 产地
    val traits: String,          // 性状
    val quality: String,         // 品质
    val images: Images,          // 图片
    val source: String,          // HKBU 源链接
    val relatedFormulas: List<String> // 关联方剂ID列表
)
```

### 4.2 方剂 (Formula)
```kotlin
data class Formula(
    val id: String,              // 唯一标识（如：F00107）
    val name: String,            // 中文名
    val pinyin: String,          // 拼音
    val englishName: String,     // 英文名
    val category: String,        // 方剂类别
    val source: String,          // 出处文献
    val ingredients: List<Ingredient>, // 组成
    val function: String,        // 功用
    val indication: String,      // 主治
    val pathogenesis: String?,   // 病机
    val usage: String,           // 用法
    val keyPoints: String?,      // 辨证要点
    val modernUsage: String?,    // 现代运用
    val precautions: String?,    // 注意事项
    val song: String?,           // 方歌
    val relatedFormulas: List<String>?, // 附方
    val herbs: List<String>      // 包含的药材ID列表
)

data class Ingredient(
    val herbId: String,          // 药材ID
    val herbName: String,        // 药材名
    val originalAmount: String?, // 原始剂量
    val modernAmount: String?    // 现代剂量
)
```

## 5. 非功能需求

### 5.1 性能
- 首屏加载 < 2s
- 搜索响应 < 500ms
- 列表滑动 60fps

### 5.2 离线支持
- 药材基础数据离线可用
- 图片懒加载 + 本地缓存

### 5.3 数据同步
- 启动时检查数据版本
- 有更新时自动同步
- 显示同步进度
