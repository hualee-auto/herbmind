# HerbMind 页面布局规范

本文档定义 HerbMind 各页面的布局规范，包含结构、间距和响应式适配。

---

## 1. 布局基础

### 1.1 安全区域

```kotlin
// 页面基础容器
@Composable
fun PageLayout(
    content: @Composable () -> Unit
) {
    Box(
        modifier = Modifier
            .fillMaxSize()
            .background(HerbColors.RicePaper)
            .padding(horizontal = HerbTokens.Space.pageMarginMobile)
    ) {
        content()
    }
}
```

### 1.2 网格系统

**手机 (320-480dp)**:
- 列数: 1
- 边距: 16dp
- 间距: 12dp

**平板 (480-768dp)**:
- 列数: 2
- 边距: 24dp
- 间距: 16dp

**大平板 (>768dp)**:
- 列数: 3
- 边距: 32dp
- 间距: 20dp

## 8. 响应式适配

### 8.1 断点定义

```kotlin
@Composable
fun ResponsiveLayout(
    content: @Composable (columns: Int) -> Unit
) {
    val configuration = LocalConfiguration.current
    val screenWidth = configuration.screenWidthDp

    val columns = when {
        screenWidth < 480 -> 1
        screenWidth < 768 -> 2
        else -> 3
    }

    content(columns)
}
```

### 8.2 网格适配

```kotlin
@Composable
fun AdaptiveCategoryGrid(
    categories: List<HerbCategory>,
    onCategoryClick: (HerbCategory) -> Unit
) {
    val configuration = LocalConfiguration.current
    val screenWidth = configuration.screenWidthDp

    val columns = when {
        screenWidth < 480 -> 3
        screenWidth < 768 -> 4
        else -> 6
    }

    LazyVerticalGrid(
        columns = GridCells.Fixed(columns),
        horizontalArrangement = Arrangement.spacedBy(12.dp),
        verticalArrangement = Arrangement.spacedBy(12.dp)
    ) {
        items(categories) { category ->
            CategoryCard(
                category = category,
                onClick = { onCategoryClick(category) }
            )
        }
    }
}
```

### 8.3 边距适配

```kotlin
@Composable
fun ResponsivePadding(
    content: @Composable () -> Unit
) {
    val configuration = LocalConfiguration.current
    val screenWidth = configuration.screenWidthDp

    val horizontalPadding = when {
        screenWidth < 480 -> 16.dp
        screenWidth < 768 -> 24.dp
        else -> 32.dp
    }

    Box(
        modifier = Modifier.padding(horizontal = horizontalPadding)
    ) {
        content()
    }
}
```

## 9. 布局组件代码

### 9.1 页面容器

```kotlin
@Composable
fun PageContainer(
    modifier: Modifier = Modifier,
    contentPadding: PaddingValues = PaddingValues(
        horizontal = HerbTokens.Space.pageMarginMobile
    ),
    content: @Composable () -> Unit
) {
    Box(
        modifier = modifier
            .fillMaxSize()
            .background(HerbColors.RicePaper)
            .padding(contentPadding)
    ) {
        content()
    }
}
```

### 9.2 区块容器

```kotlin
@Composable
fun SectionContainer(
    title: String,
    modifier: Modifier = Modifier,
    action: @Composable (() -> Unit)? = null,
    content: @Composable () -> Unit
) {
    Column(modifier = modifier.padding(vertical = HerbTokens.Space.lg)) {
        SectionTitle(title = title, action = action)
        Spacer(modifier = Modifier.height(HerbTokens.Space.md))
        content()
    }
}
```

### 9.3 网格容器

```kotlin
@Composable
fun HerbGrid(
    items: List<Herb>,
    columns: Int = 3,
    modifier: Modifier = Modifier,
    onItemClick: (Herb) -> Unit
) {
    Column(
        modifier = modifier,
        verticalArrangement = Arrangement.spacedBy(HerbTokens.Space.md)
    ) {
        items.chunked(columns).forEach { rowItems ->
            Row(
                modifier = Modifier.fillMaxWidth(),
                horizontalArrangement = Arrangement.spacedBy(HerbTokens.Space.md)
            ) {
                rowItems.forEach { item ->
                    HerbGridItem(
                        herb = item,
                        onClick = { onItemClick(item) },
                        modifier = Modifier.weight(1f)
                    )
                }
                repeat(columns - rowItems.size) {
                    Spacer(modifier = Modifier.weight(1f))
                }
            }
        }
    }
}
```
