# 隐私政策

**本草记（HerbMind）**

最后更新日期: 2026-03-20

---

## 一、概述

本草记（以下简称"本应用"）是一款中药材查询与学习工具，由 Hua Lee 开发并运营。我们非常重视您的隐私，本政策将向您说明本应用如何处理您的相关信息。

## 二、我们不收集的信息

本应用**不收集**以下任何个人信息：

- 姓名、手机号码、电子邮件地址或其他身份标识信息
- 地理位置信息
- 相机、麦克风或其他传感器数据
- 通讯录、短信或通话记录
- 设备唯一标识符（如 IMEI、广告 ID 等）
- 崩溃报告或行为分析数据

本应用**不含**任何第三方广告 SDK、数据分析 SDK 或崩溃收集 SDK。

## 三、本地存储的信息

为提供正常使用功能，本应用会在您的设备本地存储以下信息：

| 信息类型 | 说明 | 存储位置 |
|---------|------|---------|
| 中药材与方剂知识数据 | 从公开数据源同步的药典内容 | 设备本地数据库 |
| 收藏记录 | 您收藏的药材列表 | 设备本地数据库 |
| 搜索历史 | 您在应用内的搜索词记录 | 设备本地数据库 |
| 浏览历史 | 您查看过的药材或方剂记录 | 设备本地数据库 |
| 学习与复习记录 | 学习进度和复习情况 | 设备本地数据库 |

**以上所有信息仅存储在您的设备本地，不会上传至任何服务器。** 卸载应用后，上述数据将被完全删除。

## 四、网络访问

本应用需要访问网络以获取最新的中药材知识数据。网络请求仅用于：

- 从 GitHub 或 CDN 下载最新的中药材、方剂数据（单向下载，不上传任何数据）
- 加载中药材图片资源

本应用不会向任何服务器发送您的个人信息或使用行为数据。

## 五、通知权限

本应用可能会申请通知权限，用于向您发送复习提醒。通知内容仅为学习提醒，不包含任何个人信息。您可以随时在设备系统设置中关闭通知。

## 六、儿童隐私

本应用不针对 13 岁以下儿童，也不有意收集任何儿童的个人信息。

## 七、隐私政策变更

如本政策发生变更，我们将在应用或官方页面更新政策内容及日期。继续使用本应用即表示您接受更新后的政策。
