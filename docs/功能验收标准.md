## 1. 搜索功能验收标准

### 1.1 功效反查搜索

| 验收项 | 验收标准 | 优先级 |
|-------|---------|--------|
| 关键词搜索 | 支持输入功效关键词（如"补气"、"活血"） | P0 |
| 组合查询 | 支持空格分隔的多关键词组合（如"补气 健脾"） | P0 |
| 匹配度评分 | 结果按匹配度排序，显示百分比（0-100%） | P0 |
| 高亮显示 | 匹配度 ≥90% 高亮显示，≥70% 正常显示 | P1 |
| 响应时间 | 搜索响应时间 < 500ms（本地数据库） | P1 |
| 拼音搜索 | 支持拼音模糊搜索（如"huoxue"匹配"活血"） | P1 |
| 别名搜索 | 支持药材别名搜索 | P2 |
| 搜索历史 | 保存最近 10 条搜索记录，支持清空 | P2 |
| 热门标签 | 首页显示热门功效快捷标签 | P2 |

### 1.2 搜索结果展示

| 验收项 | 验收标准 | 优先级 |
|-------|---------|--------|
| 结果数量 | 显示"找到 X 味相关中药" | P0 |
| 卡片信息 | 显示药名、功效摘要、匹配度、记忆口诀 | P0 |
| 匹配功效 | 显示匹配到的具体功效 | P1 |
| 空状态 | 无结果时显示友好提示和搜索建议 | P1 |
| 点击跳转 | 点击卡片跳转到药材详情页 | P0 |

## 11. 搜索历史功能

### 11.1 需求背景
用户在搜索药材时，经常需要重复查询相似的功效组合。搜索历史功能可以帮助用户快速访问之前的查询，提升使用效率。

### 11.2 用户故事
**US-002** 作为用户，我希望查看和管理搜索历史，以便快速重复之前的查询。

### 11.3 验收标准

| 验收项 | 验收标准 | 优先级 |
|-------|---------|--------|
| 历史记录 | 自动保存最近 20 条搜索记录 | P0 |
| 记录内容 | 保存搜索关键词、搜索时间 | P1 |
| 历史展示 | 搜索页显示历史记录列表 | P0 |
| 快速搜索 | 点击历史记录直接执行搜索 | P0 |
| 单条删除 | 支持左滑或长按删除单条记录 | P1 |
| 清空历史 | 支持一键清空所有历史记录 | P1 |
| 去重处理 | 相同关键词只保留最近一条 | P2 |
| 空状态 | 无历史记录时隐藏该模块 | P1 |

## 12. 药物对比功能

### 12.1 需求背景
中医药学习中有许多功效相似但应用不同的药材（如麻黄与桂枝、当归与川芎），用户需要对比它们的功效差异、主治区别，以便准确记忆和临床应用。

### 12.2 用户故事
**US-005** 作为中医学生，我希望对比功效相似的药材，以便清晰区分它们的差异。

### 12.3 验收标准

#### 12.3.1 对比入口

| 验收项 | 验收标准 | 优先级 |
|-------|---------|--------|
| 详情页入口 | 药材详情页显示"易混淆药物"模块 | P0 |
| 对比按钮 | 点击易混淆药物进入对比页面 | P0 |
| 搜索对比 | 搜索结果支持多选后对比 | P2 |

#### 12.3.2 对比页面

| 验收项 | 验收标准 | 优先级 |
|-------|---------|--------|
| 并排展示 | 两味药并排对比展示 | P0 |
| 对比维度 | 对比：药名、分类、性味、归经、功效、主治、记忆要点 | P0 |
| 差异高亮 | 差异项使用不同背景色高亮 | P1 |
| 相同折叠 | 完全相同项可折叠收起 | P2 |
| 图片对比 | 显示两味药的药材图片 | P2 |
| 切换对比 | 支持更换对比的另一味药 | P2 |

#### 12.3.3 对比数据

| 验收项 | 验收标准 | 优先级 |
|-------|---------|--------|
| 预设对比 | 数据库预设常见易混淆药对 | P1 |
| 智能推荐 | 根据相似功效推荐对比对象 | P2 |
