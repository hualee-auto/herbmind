# 本草记 (HerbMind) - UI 视觉设计图

## 设计概述

本草记采用「新中式」设计风格，融合传统水墨意境与现代极简主义。设计强调留白、呼吸感和专业可信的学习氛围。

**设计关键词**: 水墨、竹韵、留白、呼吸感、专业

---

## 动效规范

### 页面转场
```
进入:
- 动画: 从右向左滑入
- 持续时间: 300ms
- 缓动: ease-out

退出:
- 动画: 从左向右滑出
- 持续时间: 250ms
- 缓动: ease-in
```

### 列表加载
```
动画: 渐显 + 上移
持续时间: 400ms
交错延迟: 50ms (每项)
缓动: ease-out
```

### 卡片交互
```
Hover:
- 位移: translateY(-4dp)
- 阴影: 提升一级
- 持续时间: 250ms

点击:
- 缩放: scale(0.98)
- 持续时间: 100ms
```

### 收藏动画
```
收藏:
- 图标: 缩放弹跳 1 → 1.3 → 1
- 颜色: #9E9E9E → #E53935
- 持续时间: 400ms
- 缓动: ease-bounce
```

## 响应式适配

### 断点定义
| 断点 | 宽度 | 设备 |
|------|------|------|
| Mobile | < 480dp | 手机 |
| Tablet | 480-768dp | 平板 |

### 适配规则
- **手机**: 单列布局, 16dp边距
- **平板**: 双列布局, 24dp边距, 分类网格4列

## 资源规范

### 图标
- 风格: 线性图标 (Outline)
- 线宽: 1.5dp - 2dp
