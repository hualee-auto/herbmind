# 本草记 (HerbMind) - UI 设计规范

## 设计概述

本草记是一款面向中医学习者和爱好者的中药查询应用。设计采用「新中式」风格，融合传统水墨意境与现代极简主义，营造宁静、专业、可信的学习氛围。

**设计关键词**: 水墨、竹韵、留白、呼吸感、专业
