# 本草记 (HerbMind) - 前端界面原型描述 v2

## 产品定位调整

**第一版核心**：功效反查中药 + 每日推荐  
**目标**：做「中药界的词典 + 每日一读」，先解决查询痛点，再培养使用习惯

## 8. 页面结构总结

| 页面 | 功能 |
|-----|------|
| 启动页 | 品牌展示，进入应用 |
| 首页 | 搜索入口 + 今日推荐 + 分类快捷入口 |
| 搜索页 | 输入功效，查看搜索提示 |
| 搜索结果 | 展示匹配中药，支持筛选和对比 |
| 对比模式 | 多味药并排对比 |
| 中药详情 | 单味药的完整信息 |
| 每日推荐 | 当日推荐中药的详细介绍 |
| 中药库 | 按分类浏览全部中药 |
| 收藏 | 用户收藏的中药列表 |
