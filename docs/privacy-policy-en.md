# Privacy Policy

**HerbMind**

Last Updated: 2026-03-20

---

## 1. Overview

HerbMind is a traditional Chinese medicine (TCM) herb reference and learning application developed and operated by Hua Lee. We take your privacy seriously. This policy explains how the application handles information related to you.

## 2. Information We Do NOT Collect

This application does **NOT** collect any of the following personal information:

- Name, phone number, email address, or other identifying information
- Geographic location data
- Camera, microphone, or other sensor data
- Contacts, SMS messages, or call logs
- Device unique identifiers (e.g., IMEI, advertising ID)
- Crash reports or behavioral analytics data

This application contains **no** third-party advertising SDKs, analytics SDKs, or crash reporting SDKs.

## 3. Information Stored Locally

To provide core functionality, the application stores the following information locally on your device:

| Data Type | Description | Storage Location |
|-----------|-------------|-----------------|
| Herb & formula knowledge data | Pharmacopoeia content synced from public data sources | Local device database |
| Search history | Your in-app search queries | Local device database |
| Browsing history | Herbs or formulas you have viewed | Local device database |

**All of the above information is stored exclusively on your device and is never uploaded to any server.** Uninstalling the application will permanently delete all of this data.

## 4. Network Access

The application requires network access to retrieve the latest TCM knowledge data. Network requests are used solely to:

- Download up-to-date herb and formula data from GitHub or a CDN (one-way download only — no data is uploaded)
- Load herb image resources

The application never transmits your personal information or usage behavior to any server.

## 5. Children's Privacy

This application is not directed at children under the age of 13, and we do not knowingly collect personal information from children.

## 6. Changes to This Policy

If this policy changes, we will update the content and the date on this page or within the application. Your continued use of the application following any changes constitutes your acceptance of the updated policy.
