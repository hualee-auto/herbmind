## 🎯 数据特点

### 1. 功效描述（effects）
- 简洁明了，3-5条功效
- 用顿号（、）或逗号（，）分隔
- 如：`发汗解表，宣肺平喘，利水消肿`

### 2. 记忆口诀（memoryTip）
- **七言口诀**，朗朗上口
- 押韵易记，便于背诵
- 涵盖主要功效和特点
- 示例：
  - 麻黄：`麻黄发汗又平喘，利水消肿功效全`
  - 人参：`人参大补元气好，复脉固脱安神妙`
  - 当归：`当归补血活血好，调经止痛润肠妙`

### 3. 趣味联想（association）
- 生动形象的比喻
- 帮助建立直观印象
- 便于长期记忆
- 示例：
  - 麻黄：`麻黄像个小太阳，一用就能发发汗`
  - 石膏：`石膏像块大冰块，高热烦渴降下来`
  - 金银花：`金银花黄白相间，清热解毒是花仙`

## 🚀 使用建议

### App展示顺序
1. **药材名称 + 图片**
2. **记忆口诀**（memoryTip）- 最显眼位置
3. **功效**（effects）
4. **趣味联想**（association）- 增加趣味性
5. **主治**、**用法**、**禁忌**、**考点**

### 学习场景
- **速记**：先看记忆口诀快速掌握核心功效
- **理解**：结合趣味联想建立形象记忆
- **巩固**：查看完整功效和主治加深理解
- **应试**：重点记忆考试关键点
