## 🔗 字段间关系

```
核心药材（parent）
    ├── id, name, effects...
    └── 附药（attached）
        ├── isAttached: true
        ├── parentHerb: 指向主药
        └── 自己的 effects, usage...
```

## 💡 开发提示

1. **搜索功能**: 同时搜索 `name` + `pinyin` + `aliases`
2. **图片加载**: 优先使用 `images.medicinal`（药材图）
3. **记忆模式**: 先显示 `memoryTip`，点击显示 `effects`
4. **附药处理**: 检测到 `isAttached=true` 时显示主药链接
5. **分类筛选**: 按 `category` → `subCategory` 两级筛选
