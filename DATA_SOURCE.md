# 本草记中药数据来源说明

## 数据来源

本项目中药数据主要来自以下权威来源：

### 1. 《中华人民共和国药典》（2020年版）
- **来源类型**: 国家药品标准
- **权威性**: ⭐⭐⭐⭐⭐
- **内容**: 中药的法定标准，包括性味、功效、用法用量等
- **网址**: https://ydz.chp.org.cn/

### 2. 《中药学》教材（十三五规划教材）
- **主编**: 钟赣生
- **出版社**: 中国中医药出版社
- **权威性**: ⭐⭐⭐⭐⭐
- **内容**: 中药功效分类、临床应用、配伍禁忌等

### 3. 《方剂学》教材
- **来源类型**: 中医药高等教育教材
- **权威性**: ⭐⭐⭐⭐⭐
- **内容**: 中药在方剂中的应用

### 4. 国家中医药管理局数据
- **来源类型**: 政府官方数据
- **权威性**: ⭐⭐⭐⭐⭐

## 数据字段说明

| 字段 | 说明 | 来源 |
|-----|------|-----|
| 名称 | 中药正名 | 《中国药典》 |
| 拼音 | 汉语拼音 | 《中国药典》 |
| 别名 | 常用别名 | 《中药学》教材 |
| 分类 | 功效分类 | 《中药学》教材 |
| 性味 | 四气五味 | 《中国药典》 |
| 归经 | 归经归属 | 《中国药典》+ 教材 |
| 功效 | 主要功效 | 《中国药典》 |
| 主治 | 主治病症 | 《中国药典》+ 临床应用 |
| 记忆口诀 | 自编助记口诀 | 参考各类速记资料 |
| 趣味联想 | 形象化记忆 | 自编 |
| 考试频率 | 根据历年执业药师/医师考试统计 | 试题分析 |

## 数据质量保证

1. **基础信息**: 名称、性味、功效严格依据《中国药典》
2. **分类信息**: 按照《中药学》教材功效分类法
3. **临床应用**: 参考教材及临床常用用法
4. **考试重点**: 根据历年执业药师、执业医师考试高频考点标注
